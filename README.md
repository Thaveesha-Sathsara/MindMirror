# MindMirror
An AI-powered journaling app for emotional awareness and self-reflection.

**About the Project**
MindMirror is a research-based AI application designed to help users track and understand their emotions through daily journaling. The system analyzes the emotional tone of journal entries using NLP techniques and provides personalized prompts, summaries, and emotional trend reports. The goal is to encourage self-reflection, reduce stress, and support emotional wellness in a private, ethical, and non-clinical environment.


**Key Features (Planned)**

- Write daily journal entries
- Emotion detection using NLP (GoEmotions, ISEAR datasets)
- Weekly & monthly emotional trend reports
- AI-generated prompts to encourage self-reflection
- Privacy-focused design, with no clinical diagnoses
- Aims to promote mental clarity and emotional balance

**Tech Stack (Planned)**
Python
NLP with HuggingFace/Transformers
Django or Flask (for backend)
React (for frontend)
MongoDB
Visualization libraries (Plotly, Chart.js)

**Status**
This project is currently under active development and research.
Initial data exploration and model planning are ongoing.
